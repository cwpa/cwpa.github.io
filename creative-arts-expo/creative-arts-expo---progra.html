Creative Arts Expo - Programming

Educational Program, Game Design, Robotics

Educational Program (Individual or Team Project)

  • The program must be programmed 100% by the student or team.
  • The program must be submitted as an executable file that will run on a Windows PC.
  • Code is submitted in a pdf file format for review with programming language and development tools used specified.
  • The program serves a clear educational purpose.
  • The program is original and avoids copyright violations.
  • Directions for using the program are easy to locate and understand.
  • The program is aesthetically pleaseing (colors, graphics, music etc.)

Game Design (Individual or Team Project)

  • The game is programmed 100% by the student or team.
  • The game is submitted as an executable file that will run on a Windows PC.
  • Code is submitted in a pdf file format for review with programming language and development tools used specified.
  • The game is original and avoids copyright violations.
  • The game is school appropriate (violence not permitted).
  • The objective of the game is clear.
  • Directions for playing the game appear at the start of the game and are easy to understand.
  • Game has aesthetically pleasing graphics that compliment game play.
  • Game has aesthetically pleasing sounds/music that compliment game play.

Robotics (Individual or Team Project) 

  • Robot must be built 100% by the student or team.  Can use a kit such as Legos, Boe-Bot, or Vex or can create a custom robot.
  • Robot must be programmed 100% by the student or team.
  • Robot must follow a predetermined path, which will be provided upon contest entry.
