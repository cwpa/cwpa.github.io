Extended Chords Composition

A - Compose a 32 measure piece Lead Sheet (Melody with chord names and Roman Numeral.  Do not voice the chords out.  It should have a similer look as to the analysis in Assignment #10)  Every chord must has some extension, some color.  Use the page mentioned in Assignment #5 to help you look for chord options.  Remember that you used certian chord extensions on certain Roman Numeral chords.  Not every extension belongs on every chord. Also note from you assignment #10 that the majority of the chords were some type of 7.

B - The sound of Chord Progressions using extended chord are new to a lot of students ears.  As you create them, try to think about possible ii-V-I relationships.  The progressions from Assignment #11 are great examples.  If you can play your progression on a keyboard, do so before you write of the melody.  Quite often you’ll hear some melody ideas in your head as you go along.  If you can't play them on a keyboard, use some type of chord prgression app.  My favorite is Chordbot.  This App works very similarly to Band-In-A-Box.  You can create long chord progressions with both simple and complex chord structures.  Your progression can also be saved as a MIDI or WAV file to be used in other programs.  If you have a program like GarageBand or similar, the midi files for the Melody and for the Chords, can be combined the same way we do in class.  This step is worth the time so you'll if it really works.  Trust your ear.

C - Your composition can have 1, 2 or 3 chords per measure in an A-A-B-A structure.  Eight measures for each section.  The A sections must have at least 1 Secondary Chord, V/X or viio/X, though you won’t call it a full key change.  You won’t be there long enough.  The B section is a official key change though you need to set it up with a Secondary Chord and set up a return to your Primary key at the end of that section.  Your B section may be in any of the Diatonic key options.  Remember that the B section needs to contrast with the A.

D - Be disciplined about the process and your project will be fine.  Don’t skip steps to just get it done.  You won’t like the result.  1) Set up you page.  Pick a key and a time signature.  2) Chart your progression using just Roman Numerals.  Remember ii-V-I relationshps always work.  3) Add the chord names to the progression.  4) Write a foundation note/Chord tone in each measure.  Look for ways to set up the use of non-chord tones.  5)  Remember that each chord now has 4 or more chord tones in it.  You’ll create a better melody if you don’t use the root of the chord that much. 6) Add Chordal Leaps, Passing Tones, etc. to set up a good melody that follows the rules of Melody we created in the 1st semester.  7)  If you have a keyboard or a chording app like Chordbot, play the melody with the chords for your own feedback.

E - Please submit your project using either Sibelius, NoteFlight or similar Notation Application.  I do not want a fuzzy picture of some hand written notes.  It’s very difficult to read.
