CW Music Theory Class On-Line

Music Theory, 1st Year and 2nd Year

1) Assigned - 3/23, Due - 3/27:  (No points, but needed for Assignment 2)  Go to MusicTheory.net and do the lesson on Diatonic Seventh Chords.  This website uses M7 for Major Seven.  I use mj7. Band In A Box uses maj7.  To review my notes on these chords and their fomulas, check here.

2) Assigned - 3/23, Due - 3/27: (40pts) Take the staff construction test on Extended 7th Chords, Root Position

3) Assigned - 3/23, Due - 4/13:  (No points, but needed for future 75pt test) Go to the Theory Graded Homework page and practice the listening to the ‘Popularly Used Diatonic Chord Progressions.’  Be prepared to have a listening test where you will identify these progressions when we return or at an earlier date if the 3 week period is extended. 

4) Assigned - 3/25, Due - 4/3:  (60 pts) Create a harmony for the included melody.  Remember that Classic Progression is always a good option, though not always required.  Plan on one or two chords per measure, though you can add a third chord if really necessary.  There must be at least 3 secondary chords (V7/X,  viio7/x or viio7(m7b5)/X) in your harmonic plan.  Don’t forget to give the chord name along with the Roman Numeral.

5)  Assigned - 4/1, Due - 4/7:  (40pts)  Take the Keyboard construction test on Extended 7th Chords, Root Postion

Optional Assignments:  (50pts)  1) Find a poem you like and set it to an original melody with harmony.  If it’s a long poem, you’ll need to edit it.  This song can be written for vocal solo with piano or guitar or some type of vocal ensemble.  If you don’t want to do a vocal piece, do an instrumental.  An A-A-B-A structure is always a good go to option.  Try a piano piece solo if that works for you.  Send me a completed score.  2)  (75pts)  Send me a recorded live performance of your composition.  This can be done with friends that are not all in the same location.  Ask me and I’ll coach you on how to do this.
